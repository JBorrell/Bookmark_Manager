## Bookmark Manager


```
As a user,
So that I can start using the bookmark manager,
I'd like to sign up.

As a user,
So that I can use the links I have created,
I'd like to view my list of links.

As a user,
So that I can expand my list with new and interesting content,
I'd like to be able to add links.

As a user,
So that I can discard links I don't want anymore,
I'd like to be able to remove links.

As a user,
So that I can easily organize and refer to a particular link,
I'd like to tag them.

As a user,
So that I can easily organize and refer to a particular link,
I'd like to un-tag them.

As a user,
So that I can easily find specific links,
I'd like to filter links by tag.
```
